Help

Add an entity
Clear the view. Pick a color, enter a label name and then select an available type. If selecting "regular expression", use a service such as https://regex101.com/ to thoroughly test it. Remember to save your changes.
Edit an entity
Choose an existing entity from the list. The color, name and type can be changed. Remember to save your changes.
Delete an entity
Highlight it in the lsit and choose Delete. Deleted items sent to your system's Trash/Recycle bin.
